# 能力标签词典 + BQ 题型反查

两个用途：①挖完故事给它**打标**；②看到一道 BQ，**反查**它在考哪个能力、库里哪些故事能打。

## 核心能力标签（tags / competencies 取值）

| 标签 | 它在考什么 | 典型 BQ 开头 |
|---|---|---|
| **Ownership** | 主动担责、把事做完、对结果负责 | "a time you went beyond your role" / "took initiative" |
| **Conflict Resolution** | 处理分歧、修复关系、达成一致 | "a disagreement with a coworker/manager" |
| **Influence / Persuasion** | 没有职权也能推动、说服他人 | "influenced without authority" / "convinced someone" |
| **Stakeholder Management** | 跨团队/对上/对客户协调与对齐 | "managed competing priorities across teams" |
| **Dealing with Ambiguity** | 信息不全/没人做过时如何推进 | "a time things were unclear / no clear path" |
| **Leadership** | 带人、定方向、激励、补位领导 | "led a team / project" / "motivated others" |
| **Growth Mindset / Learning** | 复盘、从反馈和失败中成长 | "biggest failure" / "received tough feedback" |
| **Drive for Results** | 高标准、抗压、把结果做出来 | "a challenging goal" / "under tight deadline" |
| **Data-Driven Decision** | 用数据/事实做判断 | "a decision based on data" |
| **Customer Obsession** | 从用户出发、为用户取舍 | "went above and beyond for a customer" |
| **Innovation / Bias for Action** | 创新、快速试错、果断行动 | "a creative solution" / "moved fast with incomplete info" |
| **Handling Failure / Resilience** | 失败、犯错、挫折后的恢复 | "a mistake you made" / "missed a deadline" |

## 反查用法（「答题」 入口）

1. 读到一道 BQ → 在上表对应到 1–2 个核心能力。
2. 去 `story-bank/_index.md` 找带这些 `competencies` 的故事。
3. 命中多个 → 选信号最强、量化最好、与目标公司最契合的那个。
4. 全未命中 → 触发「挖新故事」 现场挖。

## 打标纪律

- 一个故事**主标 1 个 + 副标 1–2 个**就好，别贴满标签（贴满 = 没重点）。
- 主标 = 这个故事最无可争议的信号；副标 = 换个开场也能打的方向。
- 失败类故事几乎都同时带 `Growth Mindset`，记得加上。
